What is JavaScript?

JavaScript is a high-level, interpreted programming language that is used for building web interfaces. Characterized as dynamic it is a key component of the web technology stack along with HTML and CSS, and it allows web pages to be interactive.

HTML, CSS and JavaScript, How do they all relate?

Building a website is like building a house. HTML is the blueprint of the house, it provides the basic structure and defines where everything goes, from the foundations, to the windows, to the walls. "Siri, play ‘Get Low’ by Lil Jon."

CSS is like the interior designer who comes in after the house is built and makes it look nice, deciding on the colors of the walls, the types of curtains to use, and what kind of carpet goes on the floor.

JavaScript, then, is like the electricity that powers all the appliances and gadgets in the house. It makes everything interactive and functional, such as turning on lights.

Explain Control Flow and Loops?

Control flow refers to the order in which statements and instructions are executed in a program. Loops are a control structure that repeats a block of code until a certain condition is met. They are used when you need to run the same code multiple times.

In a way, Control Flow is like following a recipe. The steps are laid out in a specific order, and sometimes, you have to repeat steps - A loop!

Take making this Focaccia Bread as an example. There are 8 steps which we have follow in order to make a decent loaf.

  • Make the dough
  • Stretch and fold
  • Proof the dough
  • Transfer to the pan
  • Rise the dough
  • Add toppings
  • Dimple the dough
  • Bake!

    • Step two, the Stretch and fold you take a section of the dough, stretch it up, and fold it over on itself. You then rotate the bowl 90° and do it again, for eight stretch and folds per round. This is a loop!

    What’s the DOM and how would you interact with it?

    The DOM (Document Object Model) is the interface for web documents. It represents the structure of a webpage, and allows programming languages like JavaScript to interact with it.

    An example of how you might interact with the DOM is by using the document object. Say if we wanted to get a hold of all "p" elements within a webpage you can use Chrome DevTools. In DevTools, in Console if you were to try document.querySelectorAll('p') that would provide a list of all p elements within a webpage.

    What’s the difference between accessing data from arrays and objects?

    Accessing data from arrays and objects in JavaScript is done differently. For arrays, you access data by its numerical index, starting from 0. For example, myArray[0] would access the first element of myArray.

    Objects, on the other hand, store data in key-value pairs. You access an object's data by its key, either using dot notation or bracket notation. For example, myObject.key or myObject['key'] would access the value associated with 'key' in myObject.

    What are functions and why are they helpful?

    Functions in programming are reusable pieces of code that perform specific tasks. They are helpful because they reduce repetition, make the code more readable, and divide complex tasks into simpler sub-tasks. This prevents bugs and makes code easier to maintain and update.
